# M1kE Portfolio

Personal portfolio site for M1kE; a web page builder for small Nigerian businesses, creators, and product sellers.

## What It Is

A single-page marketing site built to convert visitors into WhatsApp enquiries. No frameworks, no libraries, no JavaScript.

## Stack

- HTML
- CSS
